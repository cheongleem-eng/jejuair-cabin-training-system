# Project Blueprint: 제주항공 객실훈련팀 통합포털

## 1. 개요 (Overview)
이 애플리케이션은 제주항공 객실승무원들의 비상상황 대응 능력을 평가하고 훈련하기 위한 시스템입니다. PRAM, Interphone 조작 학습 및 상황별 롤플레잉 시나리오를 제공합니다.

## 2. 상세 구성 (Detailed Outline)

### 스타일 및 디자인 (Style & Design)
- **테마**: 제주항공 브랜드 컬러(주황색)와 어두운 배경의 다크 모드 테마.
- **레이아웃**: 모바일 퍼스트 디자인으로, 상단 헤더, 중앙 콘텐츠 영역(스크롤 가능), 하단 내비게이션 바 구성.
- **컴포넌트**: 카드형 UI, 애니메이션 효과(Blink 등), 이모지 기반 평가 버튼.
- **아이콘**: Phosphor Icons 라이브러리 사용.

### 주요 기능 (Features)
- **포털 화면**: 네 가지 주요 모듈(PRAM, Interphone, Emergency Role-Playing, Real-time Chat) 진입점 제공.
- **로그인 시스템**: 평가자/피평가자 정보 입력 및 Firebase 연동을 통한 세션 관리.
- **롤플레잉 평가**: 단계별(Phase) 시나리오 제시 및 점수 기록 기능.
- **실시간 채팅 (Open Chat)**:
  - 관리자 방 생성 및 **삭제(휴지통 아이콘)** 기능.
  - 입장 전 닉네임 및 조(1조~6조) 설정, 관리자 모드 지원.
  - **메시지 프로필 표시**: "1조: 홍길동" 또는 "관리자" 형식으로 개선.
  - **참여자 목록**: 채팅방 내 실시간 인원 확인 및 명단 보기 기능(👥 아이콘).
- **관리자 대시보드**: 전체 훈련 기록 열람 및 엑셀 다운로드 기능.
- **Firebase 연동**: 데이터 실시간 저장 및 동기화.

## 3. 현재 작업 계획 (Current Task Plan)

### 목표: CML 타이틀 텍스트 수정 및 MALFUNCTION 입력란 괘선(줄) 추가 - 완료 (Completed)
- **타이틀 텍스트 간소화**: CML 화면 상단의 "CABIN MAINTENANCE LOG 작성 연습"에서 "작성 연습"을 제외하여 깔끔한 "CABIN MAINTENANCE LOG"로 변경.
- **MALFUNCTION 입력란 개선**: 내용 입력 시 가독성을 높이기 위해 `<textarea>` 내부에 약 4줄의 노트형 가로 괘선(줄)을 추가하고, 채점 결과(성공/실패)에 따라 괘선의 색상이 조화롭게 변경되도록 고도화.

### 목표: CML 모듈 모바일 화면/글씨 크기 조절 및 가이드 모달 개선 - 완료 (Completed)
- **크기 조절 (Zoom/Scale)**: CML 화면 내 `.wrap` 컨테이너에 CSS `zoom` 속성을 동적으로 대입하여 전체적인 로그북 레이아웃의 배율 조절(35% ~ 150%) 기능 구현.
- **글씨 조절 (Font Size)**: 레이아웃 배율과 별개로 주요 텍스트 요소(입력 필드, 테이블 셀, 가이드 노트 등)의 폰트 크기만 비례적으로 확대/축소(70% ~ 150%)할 수 있는 동적 스타일시트 적용 기능 구현.
- **모바일 자동 최적화 (Auto Fit)**: 모바일 기기(너비 1000px 미만)로 접속 시, 기기 가로 너비에 맞게 자동으로 최적의 배율을 계산하여 화면에 가득 차도록 맞춰주는 자동 조절(Auto-Fit) 기능 구현.
- **환경 보존**: 설정된 배율 및 폰트 크기를 `localStorage`에 자동 보관하여 다음 방문 시에도 동일하게 유지되도록 고도화.
- **사용자 인터페이스 (UI) 및 이모티콘 제거**:
  - 화면 우측 하단에 미려한 플로팅 설정 버튼(⚙️ 아이콘) 및 반투명(Glassmorphic) 컨트롤 패널 UI 제공.
  - 헤더 제목 비행기 이모티콘(`✈️`) 제거, 📘 및 🎲 이모티콘 제거 및 영문 텍스트 한글화 (`CML 작성 가이드`, `랜덤 문제`).
  - 아이폰 등 좁은 뷰포트에서 버튼 텍스트가 세로로 쪼개지는 현상 방지 (`white-space: nowrap`, 모바일 크기 대폭 축소) 및 **LOG PAGE NO** 등 상단 정보 영역의 가로 짤림 방지 (`flex-shrink: 0`, 모바일 정렬 개선).
- **CML 가로배치 모바일 최적화**: 모바일 세로 레이아웃에서 LOG PAGE NO, HL, DATE 박스들이 길게 세로로 나열되던 현상을 가로 한 줄(Row, wrap 허용) 정렬로 개선하고 컴팩트하게 리사이징.
- **CML 작성 가이드 모달 팝업 구현**:
  - 기존의 PDF Blob 다운로드 방식 대신, Mozilla `PDF.js` 라이브러리를 CDN에서 동적 로드하여 Base64 가이드 리소스를 모달 팝업 내 `<canvas>`에 직접 페이지별 인라인 렌더링.
  - Glassmorphic 블러 처리된 세련된 배경 위에 이전(◀), 다음(▶) 버튼을 탑재하여 슬라이드 넘김 효과 연출.
  - 돋보기 줌인/줌아웃(🔍 +, 🔍 -) 기능을 기본 탑재하여 작은 스마트폰 화면에서도 디테일한 항목 조작이 용이하도록 UX 대폭 증대.

### 목표: 통합 포털 5가지 메뉴 아이콘(이모티콘) 및 디자인 조화 개선 (Portal Menu Icons & Style Harmonization) - 완료 (Completed)
- 5가지 메뉴의 아이콘을 각 기능에 부합하고 통일감 있는 Phosphor Fill 아이콘으로 일원화 (PRAM: Megaphone, Interphone: Phone Call, CML: Wrench, Emergency Role-Playing: Siren, Open Chat: Chats Circle).
- 인라인 스타일에 의해 조화가 깨지던 일부 아이콘 박스의 스타일(배경색, 텍스트 색상)을 제거하고, `style.css`에서 통일감 있는 디자인 시스템(동일한 둥글기, 박스 섀도우)으로 제어.
- 5가지 메뉴 카드가 각기 다른 매력의 조화로운 그라데이션 컬러(제주항공 주황, 하늘색, 에메랄드 그린, 긴급 빨강, 보라색)를 갖추어 시각적 균형감 부여.

### 목표: 오픈 채팅 최종 고도화 (참여자 목록, 방 삭제, 프로필 개선) - 완료 (Completed)

- 관리자 전용 방 삭제(Firestore 문서 삭제) 기능 구현.
- 채팅방 내부 실시간 참여자 수 집계 및 목록 팝업 UI 구현.

- **버전 관리**: 브라우저 캐시 문제를 방지하기 위해 정적 파일 버전을 **2.0**으로 상향 조정.

### 목표: CML 자가 학습 프로그램 모듈 연동 및 스타일 고도화 (Integrate & Modernize CML Self-Study Module) - 완료 (Completed)
- **메인 포털 연결**: 통합 포털 메인 화면의 `737-800/8 CML` 카드 클릭 시 준비 중 얼럿 대신 `CML 자가 학습 프로그램_정답_PW_X.html`을 로드하도록 수정.
- **화면 구성**: 다른 학습 모듈(PRAM, Interphone)과 동일하게 상단에 뒤로가기 및 홈 버튼을 포함한 앱 헤더 디자인을 갖춘 `screen-cml` 화면과 iframe 구조를 적용하여 일관된 사용자 인터페이스 유지.
- **리소스 최적화**: 메인 화면으로 복귀 시 iframe의 소스를 초기화하여 불필요한 백그라운드 리소스 소비 방지.
- **디자인 스타일 현대화**: 
  - **테마 & 컬러**: 제주항공 아이덴티티를 살린 오렌지(`--accent: #FF5000`) 및 프리미엄 네이비 색상을 조합하고, 부드러운 그라데이션과 슬레이트 톤의 현대적인 배경 색상 적용.
  - **유리 효과(Glassmorphic) 툴바**: 상단 툴바를 부드러운 투명도와 블러 필터(`backdrop-filter: blur`), 입체감 있는 그림자를 갖춘 고품질 헤더 스타일로 변경.
  - **컴포넌트 디테일**: 버튼들을 둥근 알약 모양(`border-radius: 10px`) 및 누를 때 반응하는 물리적 스케일 클릭 애니메이션 적용.
  - **입력 필드**: 포커스 시 부드러운 글로우 파란색 아웃라인이 생기도록 트랜지션 추가하고, 정답/오답 피드백의 박스 섀도우를 깔끔하게 다듬음.
  - **랜덤 퀴즈 패널**: 제주항공 메인 오렌지 색상 포인트 라인과 깔끔한 구조의 퀴즈 카드 인터페이스로 전면 디자인 리뉴얼.
- **CML 로그북 입체 커버 페이지 구현**:
  - **디자인 모사**: 실제 Cabin Maintenance Log 북 커버 사진을 기반으로 매트 블랙/다크 슬레이트 가죽 질감의 배경, 금색의 입체감 있는 타이틀 로고 타입, 투명 비닐 홀더에 꽂힌 기번 라벨("HL 0000"), 그리고 공식 제주항공 로고 이미지(`LOGO_CML.png`)를 소형 크기(높이 19px)로 깔끔하게 배치한 커버 화면 구현.
  - **인터랙션 및 애니메이션**: 커버에 마우스를 올리면 입체적으로 뜨는 호버 액션과 함께 클릭(터치) 시 부드럽게 위로 슬라이드 휠처럼 밀려 올라가며 실제 작성 양식 페이지를 보여주는 유기적인 전환 애니메이션 적용.
  - **화면 리셋**: 뒤로가기/홈 이동 시 iframe을 비워두어 다음 접속 시 항상 커버 페이지가 첫 화면으로 초기화되도록 상태 흐름 보장.
